# 💼 Intern Portal – Full Stack Developer Internship (Round 1)

This is a basic full stack fundraising intern dashboard built using:
- 🔧 Backend: Node.js + Express
- 🎨 Frontend: React.js

## 🚀 Features

- Dummy Login page
- Dashboard with:
  - Intern name
  - Referral code
  - Total donations
  - Static Rewards
- Static Leaderboard section
